# Max Baranov - Portfolio

High-end, single-page professional portfolio built with Astro.js, Tailwind CSS, and Phosphor Icons.

## Features

- Ultra-minimalist 'Studio' aesthetic
- Dark Mode theme with deep charcoal background (#0D0D0D)
- Bento-style grid layout
- AI-Augmented Engineering showcase
- Featured projects with Problem/Solution breakdown
- Responsive design
- Tailwind CSS for styling
- Phosphor Icons for beautiful, consistent icons

## Tech Stack

- **Astro.js** - Static site generator
- **Tailwind CSS** - Utility-first CSS framework
- **Phosphor Icons** - Beautiful icon library
- **React** - For interactive icon components

## Development

```bash
npm install
npm run dev
```

The site will be available at `http://localhost:4321`

## Build

```bash
npm run build
```

The build output will be in the `dist/` directory.

## Deployment

This site is configured for GitHub Pages. The GitHub Actions workflow will automatically deploy on push to main branch.
